Privacy Policy

This application allows users to transfer their own Spotify playlists to YouTube Music.

Data Access

The app accesses the user’s YouTube account via Google OAuth only to:

  • Create YouTube playlists
  • Add videos to playlists selected by the user

Data Storage

This app does not store any personal data, playlists, or OAuth tokens on any server.

OAuth tokens are stored temporarily in the user’s browser session and are deleted when the session ends.

Third-Party APIs

This app uses the Spotify Web API and the YouTube Data API solely to perform playlist transfer requested by the user.

User Control

Users can revoke access at any time from their Google Account security settings.
