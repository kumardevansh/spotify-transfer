Privacy Policy

This Privacy Policy explains how the application Spotify to YouTube Music Transfer accesses, uses, and handles Google user data.

Data Accessed

The application accesses the following Google user data:

  • YouTube account access via the YouTube Data API
  • Permission to create YouTube playlists
  • Permission to add videos to user-owned playlists

The application does not access:

  • Google profile information
  • Email address
  • Contacts
  • Messages
  • Any non-YouTube Google services

How the Data Is Used

Google user data is used only to:

  • Create a new YouTube playlist selected by the user
  • Search YouTube for matching videos for Spotify tracks
  • Add those videos to the playlist

The application does not use Google user data for analytics, advertising, tracking, or marketing.

Data Storage

No Google user data is stored on any server.

OAuth access tokens are stored temporarily in the user's browser session (sessionStorage) and are deleted when the session ends.

Data Sharing

The application does not share Google user data with third parties.

User Control and Revocation

Users can revoke access at any time from their Google Account security settings.

Third-Party APIs

  • Spotify Web API (playlist reading)
  • YouTube Data API v3 (playlist creation and video insertion)
