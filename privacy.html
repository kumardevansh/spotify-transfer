Privacy Policy

This Privacy Policy describes how the application Spotify to YouTube Music Transfer accesses, uses, and handles user data.

Data Accessed

The application accesses the following Google user data through Google OAuth:

  • YouTube account access via the YouTube Data API
  • Permission to create YouTube playlists
  • Permission to add videos to user-owned YouTube playlists

The application does not access:

  • User profile information
  • Email address
  • Contacts
  • Private messages
  • Any non-YouTube Google services

How the Data Is Used

Google user data is used strictly for the following purposes:

  • Creating a new YouTube playlist selected by the user
  • Searching for matching YouTube videos for Spotify tracks
  • Adding those videos to the created YouTube playlist

The application does not use Google user data for analytics, advertising, tracking, profiling, or marketing.

Data Storage

The application does not store Google user data on any server.

OAuth access tokens are stored temporarily in the user's browser session (sessionStorage) and are automatically discarded when the session ends.

No playlists, videos, tokens, or personal data are persisted or logged.

Data Sharing

The application does not share Google user data with any third parties.

User Control and Revocation

Users can revoke the application's access to their Google account at any time from the Google Account Security settings page.

Third-Party APIs

This application uses the following third-party APIs:

  • Spotify Web API (to read user playlists)
  • YouTube Data API v3 (to create playlists and add videos)
