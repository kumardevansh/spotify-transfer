Privacy Policy

This application allows users to transfer their own Spotify playlists to YouTube Music.

Data Access

  • Create YouTube playlists
  • Add videos to playlists selected by the user

Data Storage

No personal data, playlists, or OAuth tokens are stored on any server. Tokens exist only temporarily in the user’s browser session.

Third-Party APIs

This app uses the Spotify Web API and the YouTube Data API solely to perform the requested playlist transfer.

User Control

Users can revoke access at any time from their Google Account security settings.
