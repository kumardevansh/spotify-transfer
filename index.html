Failures / notes
Notes: This frontend no longer contains client IDs. You must deploy two small server endpoints: /api/auth_spotify and /api/auth_google which redirect the user to each provider's authorize URL (server uses env vars with client IDs and PKCE challenge). The server must also provide /api/exchange_google to exchange Google code for tokens.
